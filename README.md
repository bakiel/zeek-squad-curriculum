# Zeek Squad Curriculum

A comprehensive after-school coding and digital literacy programme for young learners.

## Programme Overview

Zeek Squad is designed to introduce children to computational thinking, coding fundamentals, and digital citizenship through interactive lessons and engaging games.

## Getting Started

### For Educators

1. Review the curriculum overview in `docs/curriculum-overview.md`
2. Each week's folder contains:
   - Lesson plan (PDF/Markdown)
   - Activity sheets
   - Game instructions
   - Assessment rubrics

### For Parents

See `resources/parent-guides/` for:
- Programme overview
- How to support learning at home
- FAQ

## Weekly Schedule

| Day | Focus Area |
|-----|-----------|
| Monday | Main lesson introduction |
| Tuesday | Interactive games and practice |
| Wednesday | Hands-on activities |
| Thursday | Group projects |
| Friday | Review and fun challenges |

## Technical Requirements

- Modern web browser for interactive games
- Printer for printable activities (optional)
- Internet connection for online resources
